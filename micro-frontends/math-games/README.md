# Math Games Micro Frontend

This micro frontend contains all mathematics games organized by grade level (1-8).

## Structure

```
src/
├── games/
│   ├── grade1/    # 1st grade math games
│   ├── grade2/    # 2nd grade math games
│   ├── grade3/    # 3rd grade math games
│   ├── grade4/    # 4th grade math games
│   ├── grade5/    # 5th grade math games
│   ├── grade6/    # 6th grade math games
│   ├── grade7/    # 7th grade math games
│   └── grade8/    # 8th grade math games
├── components/    # Shared math game components
├── hooks/         # Custom hooks for math games
└── types/         # TypeScript types
```

## Development

```bash
# Install dependencies
npm install

# Start dev server (port 5001)
npm run dev

# Build for production
npm run build
```
